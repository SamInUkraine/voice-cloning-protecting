# Порівняння аудіо

Це статичний сайт українською мовою для демонстрації двох аудіо:
- оригінальне: `audio/voice_clean.wav`
- захищене: `audio/voice_protected.wav`

## Як використати
1. Розпакуйте архів.
2. Замініть файли в папці `audio/` на свої реальні аудіофайли з такими самими назвами:
   - `voice_clean.wav`
   - `voice_protected.wav`
3. Якщо потрібно, замініть спектрограми в папці `assets/`:
   - `spectrogram-original.png`
   - `spectrogram-protected.png`
4. Відкрийте `index.html` у браузері або завантажте всю папку на GitHub Pages / Netlify / Vercel.

У файлах зараз лежать 18-секундні тихі placeholder-аудіо, щоб інтерфейс одразу працював.
